# Governance

## Prompt assets
Prompts are treated as governed artifacts. Each prompt must:
- Separate **System Instructions** from **User Prompt**
- Define a deterministic **Output Contract** (JSON/Table)
- Declare **data sensitivity** and **approved environment**

Use the **Prompt Review Checklist** before marking a prompt as Approved.

## Prompt Asset Register
Use `governance/Prompt_Asset_Register.xlsx` as the starter register.
**Recommendation:** store the authoritative register in controlled storage (e.g., SharePoint) and reference it here.
